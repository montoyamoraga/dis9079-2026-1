## Descripción textual del proyecto
El proyecto consiste en desarrollar un sistema IoT distribuido utilizando una Raspberry Pi, un Arduino UNO R4 WiFi, un HC-SR04 Ultrasonic Sensor y un SG90 Micro Servo Motor conectados mediante la plataforma Adafruit IO utilizando el protocolo MQTT.

La Raspberry Pi se encarga de controlar el sensor ultrasónico HC-SR04, medir la distancia de un objeto y enviar periódicamente los datos obtenidos hacia Adafruit IO a través de internet. Posteriormente, el Arduino UNO R4 WiFi consulta la información almacenada en la plataforma y controla el movimiento del servo motor SG90 según la distancia recibida.

El objetivo del proyecto es demostrar la comunicación inalámbrica entre distintos dispositivos mediante tecnologías IoT, integrando adquisición de datos físicos, transmisión en la nube y control remoto de actuadores en tiempo real. Para evitar saturar el servicio gratuito de Adafruit IO, el sistema incorpora intervalos de tiempo entre envíos de datos.
